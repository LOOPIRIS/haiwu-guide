1公司介绍

公司背景和历史

海悟初创于1995年，专注于专业场景数字能源和清洁能源解决方案及解决方案定义的产品与服务。是全国冷冻空调设备标准化技术委员会委员、中国制冷空调工业协会副理事长单位；中国通信标准化协会全权会员、中国通信企业协会理事单位；中国工程建设标准化协会副主任委员单位；英国工程技术学会（IET）会员。

公司深耕咨询、设计、联合开发、机电总包、测试认证、综合代维、优化升级全价值链垂直整合，致力于为数据中心、通信基站、供暖、储能等场景提供解决方案及产品与服务。服务于通信、互联网、电力、石油、政府、金融、教育、医疗、交通等行业客户。荣获国家级工业产品绿色设计示范企业、广东省高效节能特种空调工程技术研究中心、广东省企业技术中心等国家级、省级荣誉。

研发主要涉及系统

PDM系统

PDM的中文名称为产品数据管理（Product Data Management），是一门用来管理所有与产品相关信息（包括零件信息、配置、文档、CAD文件、结构、权限信息等）和所有与产品相关过程（包括过程定义和管理）的技术。通过实施PDM，可以提高生产效率，有利于对产品的全生命周期进行管理，加强对于文档、图纸、数据的高效利用，使工作流程规范化。

PDM操作指引查询路径 PDM → 协同管理 → 文档管理 → 02部门文档 → 01研发中心管理文档 → 06部门学习材料 → 03 PDM系统操作提示

SAP系统

SAP起源于Systems Applications and Products in Data Processing，是一款企业管理解决方案软件。SAP可用于下达、记录与查询企业发生的业务（销售订单、采购订单、生产订单、库存的出入库、人员招聘、供应商管理、客户管理等），分析企业的业务数据（资金、库存、销售、生产等多维度分析）。

SAP学习资料查询路径 PDM → 协同管理 → 文档管理 → 02部门文档 → 01研发中心管理文档 → 06部门学习材料 → 02 SAP系统学习资料

5管理制度介绍

试用期管理规定

试用期时长 试用期6个月（劳动合同签订），详见OA流程。

试用期不符合录用条件的情形

以下情形属于不符合录用条件
  • 试用期内患病或非因工负伤累计请假达15天以上，医疗期届满
  • 从其他单位离职，未依法解除、终止劳动合同或劳动关系的
  • 拖欠其他企业或单位公款尚未清偿的
  • 伪造身份证、学历、学位证书、职称证书、获奖证书、工作简历和个人信息的
  • 个人简历、入职登记表所列内容与真实情况不符的
  • 试用期满之前，无法按照公司规定办理入职手续或备齐相关资料的
  • 试用期内违反《员工纪律管理制度》应予以记小过及以上行政处分的
  • 不能按质按量完成岗位说明书要求的工作任务，或职业技能考察、考核不合格
  • 试用期内连续两个月考核得分低于60分或累计平均考核得分低于60分
  • 入职后不同意购买社会保险或因员工原因致使公司无法购买社会保险的
  • 无法提供健康入职体检证明，患有精神类疾病或法定禁止工作的传染病
  • 入职前从事接触职业病危害作业工作但无法提供离岗前职业健康检查证明
  • 拒绝公司组织的健康体检或职业健康检查的

延期转正情形

员工有以下情况之一的，延长试用考核期
  • 入职前3个月内，累计事假超过3个工作日
  • 入职前3个月内，累计有薪假（不含病假）超过5个工作日
  • 入职前3个月内，累计病假超过7个工作日
  • 入职前6个月内，有阶段考核得分为60分以下或累计平均考核得分低于60分
  • 公司或用人部门认为需要延长试用考核期的其它情形

Q2：月度绩效什么时候提交？

每月6号前提交当月月度绩效计划表，每月20号前完成上月月度绩效评估表。每月15号后新入职员工无需填写，次月启动填写。

因公务外出等客观原因无法正常签到/签退的，可进行后台考勤系统补签卡。但每月忘刷卡超过4次将给予"警告"处分。欺骗公司未出勤却申请补签卡的将给予"解雇"处分。

在OA系统搜索栏中选择"文档"，输入关键词搜索。输入"ISO9001"可查看体系文件清单；输入"RDM"可搜索研发归档文件。

Q5：研发工时在哪里填写？
